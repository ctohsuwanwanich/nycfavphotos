# New's NYC Photo Diary 📷

An interactive web map presenting my collection of photos I have taken in New York City from Fall 2025 to Spring 2026. This is all my personal favorites and recommendations around the city, with a personalized write-up for each spot. Hope you enjoy it! 

## Features
- Interactive Map
- Legend:
Blue: Daytime Spots
Pink: Sunset Spots
- Filter function at the top of the page.
